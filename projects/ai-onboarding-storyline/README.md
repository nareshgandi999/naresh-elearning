# AI-Powered Employee Onboarding Experience

## Overview

A modern onboarding module developed using Articulate Storyline 360 and AI-powered content creation tools.

This project demonstrates:
- AI-generated presenter video
- Interactive flip cards
- Scenario-based learning
- Custom navigation controls
- Modern UI design
